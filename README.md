# 📊 Анализатор продаж МДЛП - World Medicine

Комплексное веб-приложение для анализа данных МДЛП (Мониторинг движения лекарственных препаратов) с детализацией до районного уровня по Приволжскому федеральному округу.

![React](https://img.shields.io/badge/react-18.3.1-61dafb.svg)
![TypeScript](https://img.shields.io/badge/typescript-5.0+-3178c6.svg)
![License](https://img.shields.io/badge/license-Private-red.svg)

## 🎯 Основные возможности

### 📈 12 функциональных модулей:
1. **Загрузка файлов** - Drag & Drop интерфейс для импорта данных
2. **Дашборд** - Комплексная визуализация ключевых метрик
3. **Управление данными** - Работа с планами и историческими данными
4. **Проблемные зоны** - Автоматическое выявление критических областей
5. **Сравнение периодов** - Анализ год к году, по кварталам и месяцам
6. **Территориальный анализ** - Карта регионов с heat map
7. **Детализация регионов** - Drill-down до районного уровня
8. **ABC-анализ** - Классификация территорий и препаратов
9. **Сезонность** - Выявление сезонных паттернов
10. **Прогнозирование** - ML-прогнозы на основе исторических данных
11. **Контрагенты** - Анализ и рейтинг партнеров
12. **Отчеты** - Генерация и экспорт аналитических отчетов

### 🎨 Дизайн и UX
- ✨ Футуристичный дизайн в стиле Захи Хадид
- 📱 Полностью адаптивный дизайн (Desktop, Tablet, Mobile)
- 🎭 Анимации и transitions на базе Motion (Framer Motion)
- 🌙 Современный UI с градиентами и glass-morphism эффектами

### 🔐 Система пользователей
- Email-авторизация с корпоративным доменом @orney.ru
- Восстановление пароля
- Профили пользователей
- Сохранение сессии в localStorage

### 💾 Работа с данными
- Импорт/Экспорт данных (Excel, CSV, JSON)
- Архив отчетов с возможностью сохранения состояния
- История операций
- Блокировка редактирования исторических данных

## 🚀 Быстрый старт

### Требования
- Node.js 18.0 или выше
- npm или pnpm

### Установка

```bash
# Клонировать репозиторий
git clone https://github.com/yourusername/mdlp-analytics.git
cd mdlp-analytics

# Установить зависимости
npm install
# или
pnpm install

# Запустить dev-сервер
npm run dev
# или
pnpm dev
```

### Сборка для production

```bash
npm run build
# или
pnpm build
```

## 💡 Особенности реализации

- **Локальное хранилище**: Все данные сохраняются в localStorage браузера
- **Mock данные**: Реалистичные тестовые данные для демонстрации
- **Оптимизация**: Lazy loading, мемоизация, виртуализация списков
- **Accessibility**: ARIA-атрибуты, keyboard navigation
- **Performance**: Оптимизированные re-renders, debouncing
