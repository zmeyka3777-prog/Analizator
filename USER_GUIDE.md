# 📘 Руководство пользователя - Анализатор продаж МДЛП

## 🎯 Быстрый старт

### Вход в систему
1. Откройте приложение
2. Введите корпоративную почту: `вашеимя@orney.ru`
3. Введите пароль (минимум 4 символа)
4. Нажмите "Войти в систему"

### Восстановление пароля
1. На экране входа нажмите "Забыли пароль?"
2. Введите корпоративную почту
3. Нажмите "Отправить ссылку для сброса"
4. Проверьте почту (в т.ч. папку "Спам")

---

## 📊 Основные разделы

### 1. 📤 Загрузка файлов

**Назначение**: Импорт данных о продажах

**Как использовать:**
1. Перетащите файлы Excel в область загрузки
   - Или нажмите на область и выберите файлы
2. Дождитесь завершения загрузки (зеленая галочка)
3. Нажмите "Начать анализ"
4. Автоматически откроется Дашборд

**Поддерживаемые форматы:**
- ✅ Excel (.xlsx, .xls)
- ✅ CSV (.csv)
- ✅ JSON (.json)

---

### 2. 🏠 Дашборд

**Назначение**: Обзор ключевых показателей

**Основные элементы:**

#### Карточки KPI (вверху)
- **Общий объем продаж** - Сумма в рублях за период
- **Выполнение плана** - % от целевых показателей
- **Темп роста** - Динамика к прошлому периоду
- **Активные регионы** - Количество территорий с продажами

#### Фильтры
- **Препарат**: Выберите нужный препарат или "Все препараты"
- **Год**: 2023, 2024 или "Все года"
- **Период**: Год, Квартал, Месяц

#### Графики
- **График продаж** - Динамика по месяцам
- **Топ регионов** - Лучшие территории
- **Доля препаратов** - Pie chart распределения

**Полезные функции:**
- 🔄 Кнопка обновления - перезагрузка данных
- 💾 Сохранить отчет - добавить в архив
- 📄 Печать - отправка на принтер
- ⚙️ Настройки - переход к управлению данными

---

### 3. 🗄️ Управление данными и планами

**Назначение**: Работа с планами продаж и историческими данными

#### Вкладка "Исторические данные"
- Просмотр загруженных данных по периодам
- 🔒 **Закрыть данные** - блокировка от изменений
- 🔓 **Открыть данные** - разблокировка
- ⬇️ **Загрузить данные** - импорт за конкретный период

#### Вкладка "Планы продаж"
- Просмотр плановых показателей
- ✏️ **Редактировать планы** - изменение целевых значений
- 📊 Статус выполнения по каждому региону

#### Вкладка "Экспорт/Импорт"
- **Экспорт данных**: Excel, CSV, JSON
- **Импорт данных**: загрузка из файлов
- **История операций**: лог всех действий

---

### 4. ⚠️ Проблемные зоны

**Назначение**: Автоматическое выявление критических областей

**Типы проблем:**
- 🔴 **Критическое** - падение >30%
- 🟡 **Среднее** - падение 15-30%
- 🔵 **Низкое** - падение <15%

**Критерии проблемности:**
- Низкое выполнение плана (<80%)
- Отрицательная динамика продаж
- Низкая доля рынка

**Действия:**
- Кликните на проблемную зону для детализации
- Используйте рекомендации системы
- Экспортируйте список проблемных зон

---

### 5. 📈 Сравнение периодов

**Назначение**: Анализ динамики во времени

#### Режимы сравнения:

**Год к году**
- Сравнение 2024 vs 2023
- Абсолютные и относительные изменения
- Тренд по месяцам

**Кварталы**
- Q1, Q2, Q3, Q4
- Внутригодовая динамика
- Сезонные паттерны

**Месяцы**
- Помесячное сравнение
- Детальная динамика
- Выявление аномалий

**Метрики:**
- Объем продаж (руб.)
- Количество упаковок
- Средняя цена
- Доля рынка

---

### 6. 🗺️ Территориальный анализ

**Назначение**: Географическое распределение продаж

**Функции:**
- **Карта регионов** - визуализация по территориям
- **Heat Map** - цветовая индикация выполнения плана
  - 🟢 Зеленый: >100% плана
  - 🟡 Желтый: 80-100% плана
  - 🔴 Красный: <80% плана
- **Сортировка** - по объему, росту, плану
- **Фильтрация** - по проблемным зонам

**Как использовать:**
1. Выберите препарат и период
2. Кликните на регион для детализации
3. Просмотрите показатели
4. Экспортируйте отчет

---

### 7. 🔍 Детализация региона

**Назначение**: Drill-down анализ до районного уровня

**Навигация:**
- **Breadcrumbs** вверху показывают текущий путь
- Кликните на район для детализации
- Используйте кнопку "Назад" для возврата

**Уровни детализации:**
1. Федеральный округ (ПФО)
2. Регион (например, Татарстан)
3. Район (например, Казань)

**Показатели:**
- Объем продаж
- Выполнение плана
- Темп роста
- Доля в регионе

---

### 8. 📊 ABC-анализ

**Назначение**: Классификация территорий по важности

**Категории:**

**Группа A** (80% продаж)
- Ключевые территории
- Требуют максимального внимания
- Стратегические регионы

**Группа B** (15% продаж)
- Средние территории
- Потенциал роста
- Стандартная работа

**Группа C** (5% продаж)
- Небольшие территории
- Минимальные ресурсы
- Поддерживающая работа

**Матрица ABC-XYZ:**
- **X** - стабильный спрос
- **Y** - сезонные колебания
- **Z** - нестабильный спрос

**Рекомендации:**
- Система автоматически генерирует рекомендации для каждой группы

---

### 9. 📆 Сезонность

**Назначение**: Выявление сезонных паттернов

**Анализируемые показатели:**
- Индексы сезонности по месяцам
- Пиковые периоды продаж
- Спады активности

**Графики:**
- **Помесячная динамика** - bar chart
- **Тренд продаж** - line chart с сглаживанием
- **Сезонные коэффициенты** - relative values

**Применение:**
- Планирование закупок
- Оптимизация запасов
- Маркетинговые кампании

---

### 10. 🎯 Прогнозирование

**Назначение**: Построение прогнозов продаж

**Методы:**
- **Линейный тренд** - базовый прогноз
- **Сезонная декомпозиция** - учет сезонности
- **Экспоненциальное сглаживание** - сложные модели

**Сценарии:**
- 📈 **Оптимистичный** (+15% к тренду)
- 📊 **Реалистичный** (базовый тренд)
- 📉 **Пессимистичный** (-15% к тренду)

**Доверительные интервалы:**
- Показаны серой областью на графике
- 95% доверительный интервал

**Точность прогноза:**
- MAPE (Mean Absolute Percentage Error)
- Историческая точность модели

---

### 11. 🏢 Контрагенты

**Назначение**: Анализ партнеров и дистрибьюторов

**Показатели:**
- **Рейтинг** - позиция по объему продаж
- **Объем** - сумма в рублях
- **Доля** - процент от общего объема
- **Динамика** - изменение к прошлому периоду

**Категории:**
- 🥇 **ТОП-партнеры** - первая десятка
- 🥈 **Средние** - активные партнеры
- 🥉 **Малые** - небольшие объемы

**Функции:**
- Сортировка по различным критериям
- Фильтрация по категориям
- Детализация по каждому контрагенту
- Экспорт списка

---

### 12. 📄 Отчеты

**Назначение**: Генерация аналитических отчетов

**Типы отчетов:**

1. **Общий отчет**
   - Сводная информация
   - Все препараты и регионы
   - Форматы: PDF, Excel

2. **Отчет по регионам**
   - Территориальный разрез
   - Детализация до районов
   - Карты и графики

3. **Отчет по препаратам**
   - Динамика по каждому препарату
   - Доли рынка
   - Сравнительный анализ

4. **Отчет ABC-анализа**
   - Классификация территорий
   - Рекомендации
   - Матрица ABC-XYZ

**Действия с отчетами:**
- 👁️ **Предпросмотр** - просмотр перед скачиванием
- 📥 **PDF** - скачать в PDF
- 📊 **Excel** - скачать в Excel
- 📧 **Email** - отправить на почту

---

### 13. 🗄️ Архив отчетов

**Назначение**: Хранение и управление сохраненными отчетами

**Функции:**

**Сохранение отчета:**
1. В любом разделе нажмите кнопку 💾 "Сохранить отчет"
2. Введите название отчета
3. Нажмите "Сохранить"

**Загрузка отчета:**
1. Перейдите в раздел "Архив отчетов"
2. Найдите нужный отчет
3. Нажмите "Загрузить отчет"
4. Состояние восстановится (фильтры, данные)

**Удаление отчета:**
- Нажмите ❌ в правом верхнем углу карточки отчета

**Информация в карточке:**
- Название отчета
- Дата и время создания
- Примененные фильтры
- Тип отчета

## 🐛 Решение проблем

### Не загружаются данные
1. Проверьте формат файла
2. Убедитесь, что файл не поврежден
3. Попробуйте перезагрузить страницу
4. Очистите кэш браузера

### Не отображаются графики
1. Убедитесь, что данные загружены
2. Проверьте выбранные фильтры
3. Попробуйте другой препарат/период
4. Перезагрузите страницу

### Медленная работа
1. Закройте лишние вкладки браузера
2. Используйте фильтры
3. Ограничьте период анализа
4. Очистите кэш браузера

### Ошибка входа
1. Проверьте, что используете @orney.ru
2. Проверьте правильность пароля
3. Попробуйте восстановить пароль
4. Обратитесь в IT-отдел
