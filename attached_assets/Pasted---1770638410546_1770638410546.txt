Переделка визуализации: убрать переключатель "Уп./₽", показывать упаковки И деньги одновременно. Добавить круговые диаграммы и сравнение по годам.
ВАЖНО: пересчёт в рубли сейчас работает только на "По препаратам". Нужно чтобы работал ВЕЗДЕ — для этого на бэкенде для каждого региона/округа/контрагента нужна разбивка по препаратам, чтобы фронтенд мог пересчитать через прайс-лист.
1. Дашборд MDLP — полная переделка KPI-карточки "Всего продаж":
┌─────────────────────────────────────────────┐
│  ВСЕГО ПРОДАЖ                               │
│  1 021 985 уп.  │  714 523 650 ₽            │
│─────────────────────────────────────────────│
│  2024: 469 172 уп. (328 420 400 ₽)  Янв-Дек│
│  2025: 515 796 уп. (361 057 200 ₽)  Янв-Дек│  ↑ +9.9%
│  2026:  37 017 уп. ( 25 046 050 ₽)  Янв    │
└─────────────────────────────────────────────┘

Период определять автоматически из данных (все 12 мес → "Янв-Дек", один → название месяца)
Процент динамики только между полными годами (2025 vs 2024)
Рубли = сумма (количество каждого препарата × его цена из drug_prices)

2. Графики на дашборде — рядом два графика:

Левый: столбчатая диаграмма динамики продаж по месяцам в УПАКОВКАХ (линии за каждый год наложены для сравнения: 2024 синий, 2025 зелёный, 2026 красный)
Правый: такая же диаграмма в РУБЛЯХ

3. Круговые диаграммы на дашборде — два рядом:

Левая: доля каждого препарата в УПАКОВКАХ
Правая: доля каждого препарата в РУБЛЯХ
Привязать к фильтру года (при выборе 2024 — показывает только за 2024)
Топ-10 препаратов + "Остальные" серым
При наведении: название, количество, процент
Яркие различимые цвета, один препарат = один цвет везде
Использовать recharts (PieChart)

4. WM Дашборд — аналогично:

KPI карточка: упаковки и рубли рядом + разбивка по годам
Круговая по препаратам: упаковки и рубли рядом
Круговая по федеральным округам: упаковки и рубли рядом
Таблица по округам: колонки упаковки И рубли

5. Вкладка "WM По препаратам" — два блока на одном экране:

Верхняя половина: таблица и круговая в УПАКОВКАХ
Нижняя половина: таблица и круговая в РУБЛЯХ
Круговые диаграммы привязаны к фильтру года

6. Вкладка "WM По округам":

Круговая по округам в упаковках и рублях рядом
Таблица с колонками: Округ | Упаковки | Рубли

7. Остальные вкладки (Территории, Контрагенты, ABC, Проблемные зоны, Сезонность, Прогноз, Сравнение, Аналитика эффективности):

Добавить колонку "₽" рядом с "Уп." в таблицах
Круговую диаграмму где уместно (топ-10 + "Остальные")

8. Техническое для пересчёта в рубли на ВСЕХ вкладках:

Бэкенд: для каждого региона/округа/контрагента добавить в ответ объект drugBreakdown: {drugName: количество}
Фронтенд: использовать drugBreakdown для точного пересчёта (каждый препарат × свою цену)
Если drugBreakdown нет — показывать "—" в колонке рублей

9. Формат чисел:

Упаковки: 1 021 985 уп.
Рубли: 714 523 650 ₽
Большие числа: 714.5M ₽, 1.02M уп.
