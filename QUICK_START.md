## 📋 Вариант 1: Локальный запуск

### Шаг 1: Клонировать репозиторий
```bash
git clone https://github.com/yourusername/mdlp-analytics-worldmedicine.git
cd mdlp-analytics-worldmedicine
```

### Шаг 2: Установить зависимости
```bash
npm install
# или
pnpm install
```

### Шаг 3: Запустить
```bash
npm run dev
```

### Шаг 4: Открыть в браузере
```
http://localhost:5173
```

### Шаг 5: Войти
- Email: `ivanov@orney.ru` (любое имя с @orney.ru)
- Пароль: `1234` (любой, минимум 4 символа)

## 🔄 Вариант 2: Replit (Рекомендуется)

### Шаг 1: Открыть Replit
Перейти на https://replit.com

1. Нажать "+ Create Repl"
2. Выбрать "Import from GitHub"

## 🎮 Что делать дальше?

### 1. Загрузка данных
1. Кликните на область "Перетащите файлы сюда"
2. (Mock) Файлы автоматически "загрузятся"
3. Нажмите "Начать анализ"

### 2. Дашборд
- Откроется автоматически после анализа
- Посмотрите KPI карточки
- Изучите графики

### 3. Фильтры
Попробуйте изменить:
- **Препарат**: Артоксан 20мг → Лордес 5мг
- **Год**: 2024 → 2023
- **Период**: Год → Квартал

### 4. Навигация
Откройте разделы из бокового меню:
- 🗺️ Территории - карта регионов
- ⚠️ Проблемные зоны - критические области
- 📊 ABC-анализ - классификация
- 🎯 Прогноз - будущие продажи

### 5. Сохранение отчета
1. Нажмите 💾 "Сохранить отчет"
2. Введите название
3. Сохраните
4. Найдите в разделе "Архив отчетов"

## 🐛 Проблемы?

### Не запускается?
1. Проверьте версию Node.js: `node --version` (нужна 18+)
2. Переустановите зависимости: `npm ci`
3. Проверьте порт 5173 (должен быть свободен)

### Не входит?
1. Используйте email с @orney.ru
2. Пароль минимум 4 символа
3. Очистите localStorage браузера

### Не отображаются данные?
1. Загрузите файлы на странице "Загрузка"
2. Нажмите "Начать анализ"
3. Дождитесь завершения анализа
