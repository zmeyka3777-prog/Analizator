## 🚀 Шаг 1: Загрузка на GitHub (5 минут)

### Быстрый способ (GitHub Desktop)
1. Скачайте GitHub Desktop: https://desktop.github.com/
2. File → Add Local Repository → Выберите папку проекта
3. Commit to main
4. Publish repository
5. Готово! ✅

### Через командную строку
```bash
git init
git add .
git commit -m "Initial commit: MDLP Analytics v1.0.0"
git remote add origin https://github.com/YOUR_USERNAME/mdlp-analytics-worldmedicine.git
git branch -M main
git push -u origin main
```

**Подробная инструкция**: См. `GITHUB_SETUP.md`

## 🔄 Шаг 2: Импорт в Replit (2 минуты)

1. Откройте https://replit.com
2. Нажмите "+ Create Repl"
3. Выберите "Import from GitHub"
4. Вставьте URL вашего репозитория
5. Нажмите "Import from GitHub"
6. Подождите установки (2-3 минуты)
7. Нажмите "Run"
8. Готово! ✅
