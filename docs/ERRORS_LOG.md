## Правила (что НЕ делать)

1. **НЕ адаптировать код из Figma** — брать один в один, создавать недостающие зависимости
2. **НЕ использовать `any`** в TypeScript без крайней необходимости
3. **НЕ забывать `defaultValue`** для `loadFromStorage`
4. **НЕ полагаться** на одинаковые имена полей между Figma и нашими типами
5. **НЕ запускать** background агентов для работы с Figma MCP — только из основного контекста
6. **НЕ использовать `jspdf`** для кириллицы — только браузерная печать
7. **НЕ пушить и не деплоить** без явной просьбы пользователя
8. **НЕ добавлять тёмную тему** — проект только светлый
