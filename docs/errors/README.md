**Формат записи:**
```
## [YYYY-MM-DD] Краткое название ошибки
- **Статус:** открыта / исправлена / отложена
- **Где:** файл:строка или компонент
- **Симптом:** что происходит
- **Причина:** почему происходит
- **Решение:** как исправили (или почему отложили)
```
