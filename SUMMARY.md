# 📋 Краткая сводка проекта

## 📦 Анализатор продаж МДЛП v1.0.0

**World Medicine** | **Январь 2026**

---

## 🎯 Что это?

Комплексное веб-приложение для анализа данных МДЛП (Мониторинг движения лекарственных препаратов) с детализацией до районного уровня по Приволжскому федеральному округу.

---

## ✨ Ключевые возможности (в цифрах)

- 📊 **12 функциональных модулей**
- 💊 **6 анализируемых препаратов**
- 🗺️ **14 регионов ПФО**
- 🏘️ **368 районов** с детализацией
- 📈 **8+ типов графиков**
- 📱 **100% адаптивный дизайн**
- 🔐 **Email-авторизация**
- 💾 **Архив отчетов**

---

## 🛠️ Технологии

### Frontend
- React 18.3.1
- TypeScript
- Vite 6.3.5
- Tailwind CSS 4.1

### Библиотеки
- Recharts (графики)
- Lucide React (иконки)
- Motion (анимации)
- Material UI + Radix UI

## 🚀 Быстрый старт

### Локально (5 минут)
```bash
git clone https://github.com/YOUR_USERNAME/mdlp-analytics-worldmedicine.git
cd mdlp-analytics-worldmedicine
npm install
npm run dev
```
Откройте `http://localhost:5173`

### Replit (2 минуты)
1. Перейдите на https://replit.com
2. Create Repl → Import from GitHub
3. Вставьте URL репозитория
4. Нажмите Run
